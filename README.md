# 🌌 Gigantes de la Ciencia - Biografías Interactivas

Una galería curada y visualmente atractiva de las mentes más brillantes que han transformado nuestra comprensión del universo. Este proyecto busca celebrar los aportes históricos a la ciencia y promover el pensamiento crítico a través de una experiencia web interactiva.

🚀 **[Ver el sitio en vivo](https://jarrinayrton.github.io/biographies/)**

---

## ✨ Características
- **Galería de Científicos:** Tarjetas interactivas con resúmenes rápidos y aportes clave.
- **Biografías Detalladas:** Páginas dedicadas para figuras como Einstein, Curie, Tesla, y más.
- **Diseño Moderno:** Interfaz limpia, responsiva y optimizada para la lectura.
- **Tecnología:** Construido con HTML5, CSS3 (Vanilla + utilidades de estilo) y diseñado originalmente en GrapesJS.

## 👥 Científicos actualizados
Actualmente contamos con perfiles de:
- Isaac Newton
- Marie Curie
- Albert Einstein
- Nikola Tesla
- Mildred Dresselhaus
- Carl Sagan
- Erwin Schrödinger

## 📥 Descarga y Uso Local

Si deseas tener una copia de esta galería en tu computadora o usarla sin conexión:

### Opción 1: Clonar el repositorio (Recomendado para desarrolladores)
Ejecuta el siguiente comando en tu terminal:
\`\`\`bash
git clone https://github.com/jarrinayrton/biographies.git
\`\`\`

### Opción 2: Descargar como ZIP (Rápido y sencillo)
1. Haz clic en el botón verde **"Code"** en la parte superior de esta página.
2. Selecciona **"Download ZIP"**.
3. Extrae el archivo en tu computadora.

### Cómo visualizarlo
Una vez descargado, simplemente abre el archivo \`index.html\` en cualquier navegador web (Chrome, Firefox, Brave, etc.). ¡No requiere instalación de servidores externos!

## 🤝 Cómo colaborar
¡Buscamos expandir esta galería! Si quieres agregar a un científico o mejorar el diseño, sigue estos pasos:

1. **Haz un Fork** del repositorio.
2. **Crea una rama** para tu contribución (`git checkout -b feature/nueva-biografia`).
3. **Realiza tus cambios** (mira la [Guía de Contribución](CONTRIBUTING.md)).
4. **Haz un Commit** de tus cambios (`git commit -m 'Añadir biografía de Richard Feynman'`).
5. **Haz un Push** a la rama (`git push origin feature/nueva-biografia`).
6. **Abre un Pull Request**.
